🚀 آپدیت Range Scout نسخه 0.7.0 این نسخه اسکن DNS و مرحله DNSTT را خیلی منعطف‌تر کرده. حالا هم در اسکن می‌تونی بین UDP / TCP / BOTH انتخاب کنی، هم در مرحله DNSTT از UDP / TCP / DOT / DOH استفاده کنی. تنظیمات جدید هم داخل UI و فایل config کامل ذخیره و لود می‌شن. —————————————— ✨ چی عوض شده: 🔹 پروتکل اسکن DNS قابل انتخاب شد حالا اسکنر فقط UDP نیست و می‌تونی UDP، TCP یا BOTH را برای probeهای سازگاری انتخاب کنی 🔹 مرحله DNSTT حرفه‌ای‌تر شد برای تست DNSTT حالا UDP، TCP، DOT و DOH پشتیبانی می‌شن 🔹 Resolver Endpoint اضافه شد برای DNSTT می‌تونی endpoint دلخواه بدی و از placeholderهای {ip} و {port} استفاده کنی 🔹 تنظیمات جدید داخل config ذخیره می‌شن فیلدهای protocol، transport و resolverURL حالا موقع Save Config نگه داشته می‌شن و دوباره درست لود می‌شن 🔹 اسکن TCP و حالت BOTH به هسته اسکن اضافه شد نتایج resolverها حالا می‌تونن بر اساس عملکرد واقعی روی UDP و TCP ثبت بشن 🔹 تست‌ها و پایداری بهتر شد تست‌های scanner، DNSTT و embedded client گسترده‌تر شدن و رفتار transportهای جدید پوشش داده شد 🔹 رابط کاربری و نسخه برنامه آپدیت شد فیلدهای جدید DNSTT وارد UI شدن و نسخه داخل هدر روی v0.7.0 قرار گرفت ————————————— 🔗 دانلود از گیت‌هاب https://github.com/PedramMarandi/range-scout/releases/tag/v0.7.0
