📰تمامی متدهای DNS Tunneling این کانال با بهره بردن از دو سرور و مناسب کلاینت SlipNet، به‌صورت «کاملاً رایگان» در اختیار شماست. 🚨 مهم: برای اتصال به کانفیگ‌ها، نیاز به وارد کردن Username و Password دارید، که به‌صورت روزانه داخل کانال قرار خواهند گرفت. عضویت در کانال، شما رو در اولویت اتصال قرار می‌ده. نصب اپلیکیشن اندروید کلاینت SlipNet و نحوه‌ی تنظیم پارامترها 🚨 مهم: برای استفاده از کانفیگ‌ها، نیاز به DNS Resolver دارید، که با استفاده از آموزش‌های کانال، می‌تونید با توجه به نوع ارائه‌دهنده اینترنت و منطقه خود، اسکن کنید و بهترین ریزالورها رو برای استفاده در کانفیگ‌ها، به دست بیارید. آموزش نحوه‌ی اسکن DNS Resolver برای مشاهده و برداشتن کانفیگ‌ها، لینک‌های زیر را لمس کنید. 💡 نکته: لزومی به وارد کردن تمامی کانفیگ‌ها(۴٢ کانفیگ) داخل اپلیکیشن ندارید؛ می‌تونید هر رکورد رو وارد کنید، با ریزالورهایی که به دست آوردید تست کنید و اگر برای شما جواب داد، از همون استفاده کنید. 🗂 کانفیگ‌های DNSTT Socks + SSH 🗂 کانفیگ‌های NoizDNS Socks + SSH 🗂 کانفیگ‌های Slipstream Socks + SSH 🗂 کانفیگ‌های VayDNS Socks + SSH به ترتیب نوع رکورد: ▪️ VayDNS - TXT - Socks + SSH ▪️ VayDNS - CNAME - Socks + SSH ▪️ VayDNS - A - Socks + SSH ▪️ VayDNS - AAAA - Socks + SSH ▪️ VayDNS - MX - Socks + SSH ▪️ VayDNS - NS - Socks + SSH ▪️ VayDNS - SRV - Socks + SSH ▪️ VayDNS - NULL - Socks + SSH ▪️ VayDNS - CAA - Socks + SSH #DNSTT #NoizDNS #slipstream #VayDNS @break_the_barriers 💎 @xsfilterrnet 👑
